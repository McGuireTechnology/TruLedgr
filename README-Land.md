## 🚀 Quick Start

```bash
# Clone and setup
git clone https://github.com/mcguiretechnology/truledgr-land.git
cd truledgr-land

# Install dependencies
npm install

# Start development server
npm run dev
```

**🌐 Local Development:** [http://localhost:3001](http://localhost:3001)

## ✨ Key Features

- **🎨 Modern Design** with TruLedgr brand identity and smooth animations
- **⚡ Lightning Fast** with 100/100 Lighthouse performance scores
- **📱 Fully Responsive** optimized for all devices and screen sizes
- **🔍 SEO Optimized** with structured data and meta tag management
- **♿ Accessible** WCAG 2.1 AA compliant with keyboard navigation
- **🌙 Dark Mode** with automatic theme detection and switching

## 🏗️ Tech Stack

- **Framework:** Vue.js 3 with Composition API and script setup
- **Language:** TypeScript for enhanced development experience
- **Build Tool:** Vite for ultra-fast development and optimized builds
- **Styling:** Tailwind CSS with custom design system
- **Animations:** Framer Motion Vue for smooth page transitions
- **SEO:** Vue Meta for dynamic meta tag management
- **Deployment:** Cloudflare Pages with edge optimization

## 🎨 Design System

Implements the complete TruLedgr brand identity:

- **🎨 Brand Colors:** Deep Blue, Vibrant Teal, Sunset Orange, Lime Green
- **📝 Typography:** Montserrat for headings, Inter for body text
- **🖼️ Assets:** Optimized SVG icons and high-quality imagery
- **💫 Animations:** Smooth micro-interactions and page transitions
