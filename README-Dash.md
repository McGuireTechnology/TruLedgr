> 💼 **Intuitive financial dashboard** with real-time account monitoring, transaction management, and powerful analytics.

## 🚀 Quick Start

```bash
# Clone and setup
git clone https://github.com/mcguiretechnology/truledgr-dash.git
cd truledgr-dash

# Install dependencies
npm install

# Start development server
npm run dev
```

**🌐 Local Development:** [http://localhost:3000](http://localhost:3000)

## ✨ Key Features

- **📊 Real-Time Dashboard** with live balance updates and notifications
- **💳 Account Management** supporting multiple banks and account types
- **📈 Advanced Analytics** with spending insights and trend analysis
- **🔍 Smart Search** for transactions with AI-powered categorization
- **📱 Responsive Design** optimized for desktop, tablet, and mobile
- **🌙 Dark Mode Support** with automatic theme switching

## 🏗️ Tech Stack

- **Framework:** React 18 with functional components and hooks
- **Language:** TypeScript for type safety and better DX
- **Styling:** Tailwind CSS with custom TruLedgr design system
- **Build Tool:** Vite for lightning-fast development and builds
- **State Management:** Zustand for lightweight state management
- **Data Fetching:** TanStack Query for server state management
- **Charts:** Chart.js for beautiful financial visualizations

## 🎨 Design System

Built with the TruLedgr brand identity:

- **🎨 Colors:** Deep Blue (#1E2A38), Vibrant Teal (#00B8A9), Warm Gray (#F4F5F7)
- **📝 Typography:** Inter for UI, Montserrat for headings
- **🧩 Components:** Reusable design system with Storybook documentation
- **♿ Accessibility:** WCAG 2.1 AA compliant with keyboard navigation
