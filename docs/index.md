## 🏗️ Architecture Overview

TruLedgr is built with a modern microservices architecture:

- **API Server**: RESTful API built with FastAPI for high-performance operations
- **Dashboard**: React-based web interface for comprehensive financial management  
- **Landing Page**: Marketing site built with modern web technologies
- **Mobile Apps**: Native iOS and Android applications
- **Documentation**: Comprehensive guides and API references
