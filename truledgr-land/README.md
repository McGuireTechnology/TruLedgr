# TruLedgr Landing (Vue + Vite)

A minimal static landing site scaffold for TruLedgr using Vue 3 + Vite.

Quick start

```bash
cd truledgr-landing
npm install
npm run dev
```

Notes

- The site is intentionally minimal. Replace the hero illustration and copy as needed.
- For production, run `npm run build` and serve the `dist/` folder.
